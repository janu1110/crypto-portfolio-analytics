# crypto-portfolio-analytics
Crypto Portfolio Optimization and Forecasting using Python and Power BI

This project analyzes 25 cryptocurrencies to evaluate their performance, risk, and optimal portfolio allocation using Python and Power BI.

## Project Components
- Data collection and preprocessing
- Volatility and risk analysis
- CAGR and total return calculation
- Maximum drawdown analysis
- Portfolio optimization
- Bitcoin price forecasting using ARIMA
- Interactive visualization using Power BI

## Tools Used
- Python (Pandas, NumPy, Statsmodels)
- Power BI
- Data Analytics Techniques

## Files in this Repository
- crypto_analysis.py – Python script used for data analysis
- coin_metrics.csv – Cryptocurrency performance metrics
- risk_return.csv – Risk vs return analysis
- optimized_weights.csv – Optimal portfolio allocation
- equal_portfolio_growth.csv – Equal-weight portfolio performance
- optimized_portfolio_growth.csv – Optimized portfolio growth
- btc_forecast.csv – Bitcoin price forecast results
- crypto_dashboard.pbix – Power BI dashboard for visualization
